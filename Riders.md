> ### 快捷键

快速XML方法文档注释 /// 
格式化代码 Shift + Option + F
代码分区 #region #endregion 输入#re后按两次tab键
折叠代码 Command + Option + [
展开代码 Command + Option + ]
快速搜索文件 Command + P (fileName: LineNumber 快速到文件指定行)
全工程搜索 Command + Shift + F
