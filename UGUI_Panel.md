> ### Panel 容器组件介绍

Panel 容器组件又叫面板组件，实际为一个容器，可防止其他UI元素。   
当移动这个容器组件时，容器中其他UI元素也会跟着移动。使用Panel可以使整个UI结构更加清晰，且容器中还可以嵌套其他容器。   
Panel 的 Image 组件可以作为整体背景


> ### Panel 容器属性

Panel 只有一个Image组件

|	属性	|	功能	|
| :---: | :---: |
| Source Image | 目标图片 |
| Color | 颜色 |
| Material | 材质 |
| Raycast Target | 是否响应射线点击 |
| Image Type | 图片填充类型 |
